# Proyecto: Transformada de Fourier

## Objetivo

Analizar señales en el dominio del tiempo y en el dominio de la frecuencia utilizando Python y la Transformada de Fourier.

## Señales utilizadas

- Función escalón
- Pulso rectangular
- Señal senoidal

## Librerías

- NumPy
- Matplotlib

## Funcionamiento

El programa genera tres señales, calcula su Transformada de Fourier utilizando `np.fft.fft()` y muestra las gráficas del dominio del tiempo, la magnitud y la fase del espectro de frecuencia.
